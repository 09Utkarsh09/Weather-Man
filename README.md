# Weather-Man
Deployed a Simple Website that displays the current weather of any location by using  OpenWeatherMap api.
<h2> Tech Stack :</h2>
  HTML, CSS, JavaScript
<h2> API USED :</h2> 
  Open Weather API ( Link:https://openweathermap.org/api ) <br>
